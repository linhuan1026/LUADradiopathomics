# pypathomics v0.0.1
## conda install
`$ conda create --name pathomics python=3.8`

`$ conda activate pathomics`

`$ python -m pip install -r requirements.txt`

`$ cd real_pathomics`

`$ python -m pip install -e .`

## run examples

`$ python example/hello.py`
